1. Giới thiệu

Entity Framework Core là gì?

Entity Framework Core (EF Core) là ORM (Object-Relational Mapper) chính thức của Microsoft cho .NET. Thay vì viết câu lệnh SQL thủ công, lập trình viên làm việc với các class C# thuần túy — EF Core tự chuyển đổi thành SQL phù hợp.

Cách truyền thống (ADO.NET)EF Core
Viết SQL: SELECT * FROM SinhVienViết LINQ: db.SinhViens.ToList()
Phải tự map kết quả vào objectTự động map → List<SinhVien>
Thêm mới: INSERT INTO …db.SinhViens.Add(sv); db.SaveChanges();
Dễ bị SQL Injection nếu không cẩn thậnParameterized query tự động

Database-First vs Code-First

Database-FirstCode-First
Xuất phát từCSDL đã có sẵnClass C# viết trước
Lệnh dùngScaffold-DbContextAdd-Migration + Update-Database

Script SQL tạo CSDL

-- Tạo database
CREATE DATABASE QuanLySV;
GO
USE QuanLySV;
GO

-- Bảng KHOA
CREATE TABLE KHOA (
    MaKhoa  nchar(6)   NOT NULL PRIMARY KEY,
    TenKhoa nchar(50)  NOT NULL
);

-- Bảng SinhVien
CREATE TABLE SinhVien (
    MaSo      int           IDENTITY(100,1) PRIMARY KEY,
    HoTen     nvarchar(50) NOT NULL,
    NgaySinh  datetime      NULL,
    GioiTinh  bit           NOT NULL DEFAULT 1,
    DienThoai nvarchar(15) NULL,
    DiaChi    nvarchar(100) NULL,
    MaKhoa    nchar(6)      NULL REFERENCES KHOA(MaKhoa)
);

-- Bảng Mon
CREATE TABLE Mon (
    MaMh   nchar(6)  NOT NULL PRIMARY KEY,
    TenMh  nchar(50) NOT NULL,
    SoTiet int        NULL
);

-- Bảng KetQua (khóa chính kép)
CREATE TABLE KetQua (
    MaSo int      NOT NULL REFERENCES SinhVien(MaSo),
    MaMh nchar(6) NOT NULL REFERENCES Mon(MaMh),
    Diem int      NULL,
    PRIMARY KEY (MaSo, MaMh)
);

4. Scaffold – Tự động sinh Model

Scaffold là lệnh "dịch ngược" từ CSDL → class C#. Chạy một lần duy nhất trong Package Manager Console:

Scaffold-DbContext "Server=.;Database=QuanLySV;Trusted_Connection=True;TrustServerCertificate=True" Microsoft.EntityFrameworkCore.SqlServer -OutputDir Models -Force
📌 Giải thích tham số Server=. — SQL Server local (dấu chấm = localhost)
Database=QuanLySV — tên database
TrustServerCertificate=True — bỏ qua lỗi SSL (cần thiết với .NET 8)
-OutputDir Models — sinh file vào thư mục Models
-Force — ghi đè nếu đã tồn tại

5. frmMain – MDI Container

Hình 2 – Giao diện MDI (thiết kế)

Hình 3 – Ứng dụng đang chạy: 4 form con mở cùng lúc trong MDI

MDI là gì?

Multiple Document Interface (MDI) cho phép một Form cha chứa nhiều Form con bên trong. Đây là kiểu thiết kế phổ biến cho các phần mềm quản lý: mỗi chức năng mở một cửa sổ con riêng, không thoát ra ngoài ứng dụng chính.

Các thành phần kỹ thuật đặc biệt

Thành phầnVai trò
IsMdiContainer = trueBiến Form thành MDI cha, nền tự đổi màu xám
formCon.MdiParent = thisGắn Form con vào MDI cha — thiếu dòng này Form con sẽ bay ra ngoài
this.MdiChildrenMảng tất cả Form con đang mở
LayoutMdi(MdiLayout.Cascade)Sắp xếp Form con kiểu xếp tầng
MainMenuStrip = menuStrip1Khai báo MenuStrip chính để hợp nhất menu

Code frmMain.cs

using System;
using System.Windows.Forms;

namespace Lab08_QuanLySV.Models
{
    public partial class frmMain : Form
    {
        public frmMain() { InitializeComponent(); }

        // Mở Form con trong MDI — tránh mở trùng
        private void MoFormCon(Form formCon)
        {
            foreach (Form f in this.MdiChildren)
            {
                if (f.GetType() == formCon.GetType())
                {
                    f.Activate();   // đã mở → chỉ focus
                    return;
                }
            }
            formCon.MdiParent = this;  // ← quan trọng!
            formCon.Show();
        }

        // Sinh viên
        private void menuSinhVien_Click(object sender, EventArgs e) => MoFormCon(new frmSinhVien());
        private void tsbSinhVien_Click(object sender, EventArgs e)  => MoFormCon(new frmSinhVien());

        // Khoa
        private void menuKhoa_Click(object sender, EventArgs e) => MoFormCon(new frmKhoa());
        private void tsbKhoa_Click(object sender, EventArgs e)  => MoFormCon(new frmKhoa());

        // Môn học
        private void menuMonHoc_Click(object sender, EventArgs e) => MoFormCon(new frmMon());
        private void tsbMonHoc_Click(object sender, EventArgs e)  => MoFormCon(new frmMon());

        // Nhập điểm
        private void menuNhapDiem_Click(object sender, EventArgs e) => MoFormCon(new frmNhapDiem());
        private void tsbNhapDiem_Click(object sender, EventArgs e)  => MoFormCon(new frmNhapDiem());

        // Xem điểm
        private void menuKetQua_Click(object sender, EventArgs e) => MoFormCon(new frmKetQua());
        private void tsbKetQua_Click(object sender, EventArgs e)  => MoFormCon(new frmKetQua());

        // Thoát
        private void Thoat()
        {
            if (MessageBox.Show("Bạn có chắc muốn thoát?", "Xác nhận",
                MessageBoxButtons.YesNo, MessageBoxIcon.Question) == DialogResult.Yes)
                Application.Exit();
        }
        private void menuThoat_Click(object s, EventArgs e) => Thoat();
        private void tsbThoat_Click(object s, EventArgs e)   => Thoat();

        // Sắp xếp cửa sổ con
        private void menuCascade_Click(object s, EventArgs e) => LayoutMdi(MdiLayout.Cascade);
        private void menuTileH_Click(object s, EventArgs e)   => LayoutMdi(MdiLayout.TileHorizontal);
        private void menuTileV_Click(object s, EventArgs e)   => LayoutMdi(MdiLayout.TileVertical);
        private void menuDongTat_Click(object s, EventArgs e)
        {
            foreach (Form f in MdiChildren) f.Close();
        }
    }
}

Code frmSinhVien.cs

using System;
using System.Linq;
using System.Windows.Forms;
using Microsoft.EntityFrameworkCore;

namespace Lab08_QuanLySV.Models
{
    public partial class frmSinhVien : Form
    {
        private QuanLySvContext db = new QuanLySvContext();

        public frmSinhVien() { InitializeComponent(); }

        private void frmSinhVien_Load(object sender, EventArgs e)
        {
            LoadComboKhoa();
            LoadDanhSach();
        }

        private void LoadComboKhoa()
        {
            var ds = db.Khoas.Select(k => new {
                k.MaKhoa, TenKhoa = k.TenKhoa.Trim()
            }).ToList();
            cboKhoa.DataSource    = ds;
            cboKhoa.DisplayMember = "TenKhoa";
            cboKhoa.ValueMember   = "MaKhoa";
        }

        private void LoadDanhSach(string keyword = "")
        {
            var query = db.SinhViens.Include(sv => sv.MaKhoaNavigation).AsQueryable();

            if (!string.IsNullOrWhiteSpace(keyword))
            {
                string kw = keyword.Trim().ToLower();
                query = query.Where(sv => sv.HoTen.ToLower().Contains(kw)
                                      || sv.DienThoai.ToLower().Contains(kw));
            }

            var ds = query.Select(sv => new {
                sv.MaSo, sv.HoTen, sv.NgaySinh,
                GioiTinh = sv.GioiTinh ? "Nam" : "Nữ",
                sv.DienThoai, sv.DiaChi,
                TenKhoa = sv.MaKhoaNavigation.TenKhoa.Trim()
            }).OrderBy(sv => sv.HoTen).ToList();

            dgvSinhVien.DataSource = ds;
        }

        private void dgvSinhVien_SelectionChanged(object sender, EventArgs e)
        {
            if (dgvSinhVien.CurrentRow == null) return;
            lblMaSo.Text        = dgvSinhVien.CurrentRow.Cells["MaSo"].Value?.ToString();
            txtHoTen.Text       = dgvSinhVien.CurrentRow.Cells["HoTen"].Value?.ToString();
            txtDienThoai.Text   = dgvSinhVien.CurrentRow.Cells["DienThoai"].Value?.ToString();
            txtDiaChi.Text      = dgvSinhVien.CurrentRow.Cells["DiaChi"].Value?.ToString();
            string gt          = dgvSinhVien.CurrentRow.Cells["GioiTinh"].Value?.ToString() ?? "Nam";
            cboGioiTinh.SelectedItem = gt;
            if (dgvSinhVien.CurrentRow.Cells["NgaySinh"].Value is DateTime dt)
                dtpNgaySinh.Value = dt;
            // Sync ComboBox Khoa
            string tenKhoa = dgvSinhVien.CurrentRow.Cells["TenKhoa"].Value?.ToString() ?? "";
            foreach (var item in cboKhoa.Items)
            {
                var p = item.GetType().GetProperty("TenKhoa");
                if (p?.GetValue(item)?.ToString() == tenKhoa)
                { cboKhoa.SelectedItem = item; break; }
            }
        }

        private void btnThem_Click(object sender, EventArgs e)
        {
            if (string.IsNullOrWhiteSpace(txtHoTen.Text))
            { MessageBox.Show("Vui lòng nhập họ tên!", "Thông báo"); return; }

            db.SinhViens.Add(new SinhVien {
                HoTen     = txtHoTen.Text.Trim(),
                NgaySinh  = dtpNgaySinh.Value,
                GioiTinh  = cboGioiTinh.Text == "Nam",
                DienThoai = txtDienThoai.Text.Trim(),
                DiaChi    = txtDiaChi.Text.Trim(),
                MaKhoa    = cboKhoa.SelectedValue?.ToString()
            });
            db.SaveChanges();
            LoadDanhSach();
            LamMoi();
            MessageBox.Show("Thêm sinh viên thành công!", "Thông báo", MessageBoxButtons.OK, MessageBoxIcon.Information);
        }

        private void btnSua_Click(object sender, EventArgs e)
        {
            if (!int.TryParse(lblMaSo.Text, out int maSo)) return;
            SinhVien? sv = db.SinhViens.Find(maSo);
            if (sv == null) return;

            sv.HoTen     = txtHoTen.Text.Trim();
            sv.NgaySinh  = dtpNgaySinh.Value;
            sv.GioiTinh  = cboGioiTinh.Text == "Nam";
            sv.DienThoai = txtDienThoai.Text.Trim();
            sv.DiaChi    = txtDiaChi.Text.Trim();
            sv.MaKhoa    = cboKhoa.SelectedValue?.ToString();
            db.SaveChanges();
            LoadDanhSach();
            MessageBox.Show("Cập nhật thành công!", "Thông báo", MessageBoxButtons.OK, MessageBoxIcon.Information);
        }

        private void btnXoa_Click(object sender, EventArgs e)
        {
            if (!int.TryParse(lblMaSo.Text, out int maSo)) return;

            if (db.KetQuas.Any(kq => kq.MaSo == maSo))
            { MessageBox.Show("Không thể xóa! Sinh viên đã có điểm trong hệ thống.", "Lỗi", MessageBoxButtons.OK, MessageBoxIcon.Error); return; }

            if (MessageBox.Show($"Xóa sinh viên '{txtHoTen.Text}'?", "Xác nhận",
                MessageBoxButtons.YesNo) != DialogResult.Yes) return;

            SinhVien? sv = db.SinhViens.Find(maSo);
            if (sv != null) { db.SinhViens.Remove(sv); db.SaveChanges(); }
            LoadDanhSach(); LamMoi();
        }

        private void LamMoi()
        {
            lblMaSo.Text = ""; txtHoTen.Clear(); txtDienThoai.Clear(); txtDiaChi.Clear();
            cboGioiTinh.SelectedIndex = 0;
            dtpNgaySinh.Value = DateTime.Today;
            txtTimKiem.Clear(); txtHoTen.Focus();
        }
        private void btnLamMoi_Click(object s, EventArgs e) { LamMoi(); LoadDanhSach(); }
        private void btnTimKiem_Click(object s, EventArgs e) => LoadDanhSach(txtTimKiem.Text);
        private void txtTimKiem_KeyDown(object s, KeyEventArgs e)
        { if (e.KeyCode == Keys.Enter) LoadDanhSach(txtTimKiem.Text); }
    }
}

7. frmKhoa – Quản lý khoa

Đặc điểm của frmKhoa

Hình – Giao diện frmKhoa (Quản lý khoa)

  • CRUD bảng KHOA với khóa chính chuỗi (nchar 6), không phải int IDENTITY
  • Người dùng tự nhập MaKhoa → cần kiểm tra trùng trước khi thêm
  • Trước khi xóa: kiểm tra còn SinhVien thuộc khoa này không
  • Namespace: Lab08_QuanLySV.Models
namespace Lab08_QuanLySV.Models
{
    public partial class frmKhoa : Form
    {
        private QuanLySvContext db = new QuanLySvContext();

        private void frmKhoa_Load(object s, EventArgs e) => LoadDanhSach();

        private void LoadDanhSach(string kw = "")
        {
            var query = db.Khoas.AsQueryable();
            if (!string.IsNullOrWhiteSpace(kw))
                query = query.Where(k => k.TenKhoa.Contains(kw) || k.MaKhoa.Contains(kw));

            dgvKhoa.DataSource = query.Select(k => new {
                MaKhoa  = k.MaKhoa.Trim(),
                TenKhoa = k.TenKhoa.Trim()
            }).ToList();
        }

        private void dgvKhoa_SelectionChanged(object s, EventArgs e)
        {
            if (dgvKhoa.CurrentRow == null) return;
            txtMaKhoa.Text  = dgvKhoa.CurrentRow.Cells["MaKhoa"].Value?.ToString();
            txtTenKhoa.Text = dgvKhoa.CurrentRow.Cells["TenKhoa"].Value?.ToString();
        }

        private void btnThem_Click(object s, EventArgs e)
        {
            string ma = txtMaKhoa.Text.Trim().ToUpper();
            if (string.IsNullOrEmpty(ma) || string.IsNullOrWhiteSpace(txtTenKhoa.Text))
            { MessageBox.Show("Nhập đủ mã và tên khoa!"); return; }

            if (db.Khoas.Any(k => k.MaKhoa.Trim() == ma))
            { MessageBox.Show("Mã khoa đã tồn tại!", "Lỗi", MessageBoxButtons.OK, MessageBoxIcon.Error); return; }

            db.Khoas.Add(new Khoa { MaKhoa = ma, TenKhoa = txtTenKhoa.Text.Trim() });
            db.SaveChanges();
            LoadDanhSach(); LamMoi();
        }

        private void btnSua_Click(object s, EventArgs e)
        {
            string ma = txtMaKhoa.Text.Trim();
            var khoa = db.Khoas.FirstOrDefault(k => k.MaKhoa.Trim() == ma);
            if (khoa == null) return;
            khoa.TenKhoa = txtTenKhoa.Text.Trim();
            db.SaveChanges();
            LoadDanhSach();
        }

        private void btnXoa_Click(object s, EventArgs e)
        {
            string ma = txtMaKhoa.Text.Trim();

            // Kiểm tra FK trước khi xóa
            if (db.SinhViens.Any(sv => sv.MaKhoa.Trim() == ma))
            { MessageBox.Show($"Không thể xóa! Khoa '{txtTenKhoa.Text}' còn sinh viên.",
                "Không thể xóa", MessageBoxButtons.OK, MessageBoxIcon.Error); return; }

            if (MessageBox.Show($"Xóa khoa '{txtTenKhoa.Text}'?", "Xác nhận",
                MessageBoxButtons.YesNo) != DialogResult.Yes) return;

            var khoa = db.Khoas.FirstOrDefault(k => k.MaKhoa.Trim() == ma);
            if (khoa != null) { db.Khoas.Remove(khoa); db.SaveChanges(); }
            LoadDanhSach(); LamMoi();
        }

        private void LamMoi() { txtMaKhoa.Clear(); txtTenKhoa.Clear(); txtMaKhoa.Focus(); }
        private void btnLamMoi_Click(object s, EventArgs e) { LamMoi(); LoadDanhSach(); }
        private void btnTimKiem_Click(object s, EventArgs e) => LoadDanhSach(txtTimKiem.Text);
    }
}

8. frmMon – Quản lý môn học

Đặc điểm của frmMon

Hình – Giao diện frmMon (Quản lý môn học)

  • Controls nhập: txtMaMh (CharacterCasing=Upper, MaxLength=6), txtTenMh, nudSoTiet (NumericUpDown, 0–300, mặc định 45)
  • Tên property đúng: MaMh, TenMh — không phải MaMH, TenMH
  • Kiểm tra FK với bảng KetQua trước khi xóa môn
  • Dùng nudSoTiet (NumericUpDown) thay TextBox — tránh nhập sai kiểu dữ liệu
namespace Lab08_QuanLySV.Models
{
    public partial class frmMon : Form
    {
        private QuanLySvContext db = new QuanLySvContext();

        private void frmMon_Load(object s, EventArgs e) => LoadDanhSach();

        private void LoadDanhSach(string keyword = "")
        {
            var query = db.Mons.AsQueryable();
            if (!string.IsNullOrWhiteSpace(keyword))
            {
                string kw = keyword.Trim().ToLower();
                query = query.Where(m => m.TenMh.ToLower().Contains(kw)
                                      || m.MaMh.ToLower().Contains(kw));
            }
            dgvMon.DataSource = query.Select(m => new {
                MaMh  = m.MaMh.Trim(), TenMh = m.TenMh.Trim(), m.SoTiet
            }).ToList();
        }

        private void dgvMon_SelectionChanged(object s, EventArgs e)
        {
            if (dgvMon.CurrentRow == null) return;
            txtMaMh.Text  = dgvMon.CurrentRow.Cells["MaMh"].Value?.ToString();
            txtTenMh.Text = dgvMon.CurrentRow.Cells["TenMh"].Value?.ToString();
            if (int.TryParse(dgvMon.CurrentRow.Cells["SoTiet"].Value?.ToString(), out int st))
                nudSoTiet.Value = st;
        }

        private void btnThem_Click(object s, EventArgs e)
        {
            string ma = txtMaMh.Text.Trim().ToUpper();
            if (string.IsNullOrEmpty(ma) || string.IsNullOrWhiteSpace(txtTenMh.Text))
            { MessageBox.Show("Nhập đủ mã và tên môn học!"); return; }

            if (db.Mons.Any(m => m.MaMh.Trim() == ma))
            { MessageBox.Show("Mã môn đã tồn tại!", "Lỗi", MessageBoxButtons.OK, MessageBoxIcon.Error); return; }

            db.Mons.Add(new Mon { MaMh = ma, TenMh = txtTenMh.Text.Trim(), SoTiet = (int)nudSoTiet.Value });
            db.SaveChanges();
            LoadDanhSach(); LamMoi();
            MessageBox.Show("Thêm môn học thành công!", "Thông báo", MessageBoxButtons.OK, MessageBoxIcon.Information);
        }

        private void btnSua_Click(object s, EventArgs e)
        {
            string ma = txtMaMh.Text.Trim();
            Mon? mon = db.Mons.FirstOrDefault(m => m.MaMh.Trim() == ma);
            if (mon == null) return;
            mon.TenMh = txtTenMh.Text.Trim();
            mon.SoTiet = (int)nudSoTiet.Value;
            db.SaveChanges();
            LoadDanhSach();
        }

        private void btnXoa_Click(object s, EventArgs e)
        {
            string ma = txtMaMh.Text.Trim();

            // Kiểm tra FK: môn đã có điểm chưa?
            if (db.KetQuas.Any(kq => kq.MaMh.Trim() == ma))
            { MessageBox.Show($"Không thể xóa! Môn '{txtTenMh.Text}' đã có điểm.",
                "Không thể xóa", MessageBoxButtons.OK, MessageBoxIcon.Error); return; }

            if (MessageBox.Show($"Xóa môn '{txtTenMh.Text}'?", "Xác nhận",
                MessageBoxButtons.YesNo) != DialogResult.Yes) return;

            Mon? mon = db.Mons.FirstOrDefault(m => m.MaMh.Trim() == ma);
            if (mon != null) { db.Mons.Remove(mon); db.SaveChanges(); }
            LoadDanhSach(); LamMoi();
        }

        private void LamMoi()
        { txtMaMh.Clear(); txtTenMh.Clear(); nudSoTiet.Value = 45; txtTimKiem.Clear(); txtMaMh.Focus(); }

        private void btnLamMoi_Click(object s, EventArgs e) { LamMoi(); LoadDanhSach(); }
        private void btnTimKiem_Click(object s, EventArgs e) => LoadDanhSach(txtTimKiem.Text);
        private void txtTimKiem_KeyDown(object s, KeyEventArgs e)
        { if (e.KeyCode == Keys.Enter) LoadDanhSach(txtTimKiem.Text); }
    }
}

9. frmNhapDiem – Nhập điểm

Đặc điểm thiết kế

Hình – Giao diện frmNhapDiem (Nhập điểm)

📝 Controls nhập liệu

  • cboSinhVien — hiển thị "HoTen — TenKhoa"
  • cboMonHoc — hiển thị "TenMh (SoTiet tiết)"
  • nudDiem — NumericUpDown 0–10

🔘 4 nút tách biệt rõ ràng

  • btnLuuDiem (xanh) — chỉ thêm mới
  • btnCapNhat (cam) — chỉ cập nhật
  • btnXoaDiem (đỏ) — xóa + confirm
  • btnLamMoi (xám) — reset form
⚠️ Navigation propertyBảng KetQua dùng: kq.MaMhNavigation (chữ h thường), kq.MaSoNavigation. Trường khóa ngoại: kq.MaMh. Khi Insert: MaMh = maMH.PadRight(6) vì cột là nchar(6) cố định độ dài.
namespace Lab08_QuanLySV.Models
{
    public partial class frmNhapDiem : Form
    {
        private QuanLySvContext db = new QuanLySvContext();

        private void frmNhapDiem_Load(object s, EventArgs e)
        { LoadComboSinhVien(); LoadComboMonHoc(); LoadDanhSachDiem(); }

        private void LoadComboSinhVien()
        {
            var ds = db.SinhViens.Include(sv => sv.MaKhoaNavigation)
                .Select(sv => new {
                    sv.MaSo,
                    HienThi = sv.HoTen + " — " + sv.MaKhoaNavigation.TenKhoa.Trim()
                }).ToList();
            cboSinhVien.DataSource = ds;
            cboSinhVien.DisplayMember = "HienThi";
            cboSinhVien.ValueMember   = "MaSo";
        }

        private void LoadComboMonHoc()
        {
            var ds = db.Mons.Select(m => new {
                MaMh    = m.MaMh.Trim(),
                HienThi = m.TenMh.Trim() + " (" + m.SoTiet + " tiết)"
            }).ToList();
            cboMonHoc.DataSource = ds;
            cboMonHoc.DisplayMember = "HienThi";
            cboMonHoc.ValueMember   = "MaMh";
        }

        private void LoadDanhSachDiem(string keyword = "")
        {
            var query = db.KetQuas
                .Include(kq => kq.MaSoNavigation)
                .Include(kq => kq.MaMhNavigation)
                .AsQueryable();

            if (!string.IsNullOrWhiteSpace(keyword))
            {
                string kw = keyword.Trim().ToLower();
                query = query.Where(kq =>
                    kq.MaSoNavigation.HoTen.ToLower().Contains(kw) ||
                    kq.MaMhNavigation.TenMh.ToLower().Contains(kw));
            }

            dgvDiem.DataSource = query.Select(kq => new {
                kq.MaSo,
                HoTen   = kq.MaSoNavigation.HoTen,
                TenMh   = kq.MaMhNavigation.TenMh.Trim(),
                kq.Diem,
                XepLoai = kq.Diem >= 9 ? "Xuất sắc"
                        : kq.Diem >= 7 ? "Giỏi"
                        : kq.Diem >= 5 ? "Trung bình" : "Yếu"
            }).OrderBy(x => x.HoTen).ToList();
        }

        // Thêm mới điểm
        private void btnLuuDiem_Click(object s, EventArgs e)
        {
            int    maSo = (int)cboSinhVien.SelectedValue;
            string maMh = cboMonHoc.SelectedValue.ToString()!.Trim();

            if (db.KetQuas.Any(kq => kq.MaSo == maSo && kq.MaMh.Trim() == maMh))
            { MessageBox.Show("Sinh viên đã có điểm môn này!\nDùng 'Cập nhật' để thay đổi.",
                "Đã tồn tại", MessageBoxButtons.OK, MessageBoxIcon.Warning); return; }

            db.KetQuas.Add(new KetQua {
                MaSo = maSo,
                MaMh = maMh.PadRight(6),   // nchar(6) cố định độ dài
                Diem = (int)nudDiem.Value
            });
            db.SaveChanges();
            LoadDanhSachDiem(txtTimKiem.Text);
            MessageBox.Show("Lưu điểm thành công!", "Thông báo", MessageBoxButtons.OK, MessageBoxIcon.Information);
        }

        // Cập nhật điểm
        private void btnCapNhat_Click(object s, EventArgs e)
        {
            int    maSo = (int)cboSinhVien.SelectedValue;
            string maMh = cboMonHoc.SelectedValue.ToString()!.Trim();
            KetQua? kq = db.KetQuas.FirstOrDefault(k => k.MaSo == maSo && k.MaMh.Trim() == maMh);
            if (kq == null)
            { MessageBox.Show("Không tìm thấy bản ghi! Dùng 'Lưu điểm' để thêm mới."); return; }
            kq.Diem = (int)nudDiem.Value;
            db.SaveChanges();
            LoadDanhSachDiem(txtTimKiem.Text);
        }

        // Xóa điểm
        private void btnXoaDiem_Click(object s, EventArgs e)
        {
            int    maSo   = (int)cboSinhVien.SelectedValue;
            string maMh   = cboMonHoc.SelectedValue.ToString()!.Trim();
            string tenSV  = cboSinhVien.Text.Split('—')[0].Trim();
            string tenMon = cboMonHoc.Text.Split('(')[0].Trim();

            if (MessageBox.Show($"Xóa điểm của '{tenSV}' môn '{tenMon}'?", "Xác nhận",
                MessageBoxButtons.YesNo) != DialogResult.Yes) return;

            KetQua? kq = db.KetQuas.FirstOrDefault(k => k.MaSo == maSo && k.MaMh.Trim() == maMh);
            if (kq != null) { db.KetQuas.Remove(kq); db.SaveChanges(); }
            LoadDanhSachDiem(); LamMoi();
        }

        private void LamMoi()
        {
            if (cboSinhVien.Items.Count > 0) cboSinhVien.SelectedIndex = 0;
            if (cboMonHoc.Items.Count   > 0) cboMonHoc.SelectedIndex   = 0;
            nudDiem.Value = 5; txtTimKiem.Clear();
        }
        private void btnLamMoi_Click(object s, EventArgs e) { LamMoi(); LoadDanhSachDiem(); }
        private void btnTimKiem_Click(object s, EventArgs e) => LoadDanhSachDiem(txtTimKiem.Text);
        private void txtTimKiem_KeyDown(object s, KeyEventArgs e)
        { if (e.KeyCode == Keys.Enter) LoadDanhSachDiem(txtTimKiem.Text); }
    }
}

10. frmKetQua – Xem điểm & Xuất CSV

Đặc điểm của frmKetQua

Hình – Giao diện frmKetQua (Xem điểm)

🔍 Xem kết quả

  • Chọn sinh viên từ cboSinhVien
  • Nhấn btnXem → load điểm của SV đó
  • Tính ĐTB + xếp loại tự động
  • Màu xếp loại thay đổi theo kết quả

📤 Xuất CSV

  • BOM UTF-8 (\uFEFF) để Excel hiện tiếng Việt
  • Tên file tự động: KetQua_{tenSV}_{yyyyMMdd}.csv
  • Dòng tổng kết ĐTB cuối file
  • SaveFileDialog mở thư mục Desktop

Logic xếp loại và màu sắc

ĐTBXếp loạiMàu hiển thị
≥ 9.0Xuất sắcDarkOrange
≥ 7.0GiỏiDarkGreen
≥ 5.0Trung bìnhDarkBlue
< 5.0YếuDarkRed
namespace Lab08_QuanLySV.Models
{
    public partial class frmKetQua : Form
    {
        private QuanLySvContext db = new QuanLySvContext();

        private void frmKetQua_Load(object s, EventArgs e)
        {
            var ds = db.SinhViens.Include(sv => sv.MaKhoaNavigation)
                .Select(sv => new {
                    sv.MaSo,
                    HienThi = sv.HoTen + " — " + sv.MaKhoaNavigation.TenKhoa.Trim()
                }).ToList();
            cboSinhVien.DataSource    = ds;
            cboSinhVien.DisplayMember = "HienThi";
            cboSinhVien.ValueMember   = "MaSo";
        }

        private void btnXem_Click(object s, EventArgs e)
        {
            if (cboSinhVien.SelectedValue == null) return;
            int maSo = (int)cboSinhVien.SelectedValue;

            var ds = db.KetQuas
                .Include(kq => kq.MaMhNavigation)
                .Where(kq => kq.MaSo == maSo)
                .Select(kq => new {
                    TenMh   = kq.MaMhNavigation.TenMh.Trim(),
                    SoTiet  = kq.MaMhNavigation.SoTiet,
                    kq.Diem,
                    XepLoai = kq.Diem >= 9 ? "Xuất sắc"
                            : kq.Diem >= 7 ? "Giỏi"
                            : kq.Diem >= 5 ? "Trung bình" : "Yếu"
                }).OrderBy(x => x.TenMh).ToList();

            dgvKetQua.DataSource = ds;

            if (ds.Count > 0)
            {
                double dtb = ds.Average(d => d.Diem ?? 0);
                string xepLoai; Color mauSac;

                if      (dtb >= 9.0) { xepLoai = "Xuất sắc";   mauSac = Color.DarkOrange; }
                else if (dtb >= 7.0) { xepLoai = "Giỏi";        mauSac = Color.DarkGreen;  }
                else if (dtb >= 5.0) { xepLoai = "Trung bình";  mauSac = Color.DarkBlue;   }
                else                  { xepLoai = "Yếu";         mauSac = Color.DarkRed;    }

                lblDTB.Text          = $"ĐTB: {dtb:F2}";
                lblSoMon.Text        = $"Môn đã học: {ds.Count}";
                lblXepLoai.Text      = $"Xếp loại: {xepLoai}";
                lblXepLoai.ForeColor = mauSac;
            }
        }

        private void btnXuatCSV_Click(object s, EventArgs e)
        {
            if (dgvKetQua.Rows.Count == 0)
            { MessageBox.Show("Chưa có dữ liệu! Nhấn 'Xem kết quả' trước."); return; }

            using SaveFileDialog sfd = new()
            {
                Filter   = "CSV files (*.csv)|*.csv",
                FileName = $"KetQua_{cboSinhVien.Text.Split('—')[0].Trim()}_{DateTime.Now:yyyyMMdd}.csv",
                InitialDirectory = Environment.GetFolderPath(Environment.SpecialFolder.Desktop)
            };
            if (sfd.ShowDialog() != DialogResult.OK) return;

            using StreamWriter sw = new(sfd.FileName, false, System.Text.Encoding.UTF8);
            sw.Write("\uFEFF");                           // BOM UTF-8 cho Excel
            sw.WriteLine("Môn học,Số tiết,Điểm,Xếp loại");

            foreach (DataGridViewRow row in dgvKetQua.Rows)
            {
                if (row.IsNewRow) continue;
                sw.WriteLine($"{row.Cells["TenMh"].Value},{row.Cells["SoTiet"].Value},{row.Cells["Diem"].Value},{row.Cells["XepLoai"].Value}");
            }

            sw.WriteLine();
            sw.WriteLine($"Sinh viên: {cboSinhVien.Text}");
            sw.WriteLine($"{lblDTB.Text},{lblSoMon.Text},{lblXepLoai.Text}");

            MessageBox.Show($"Xuất file thành công!\n{sfd.FileName}", "Hoàn thành",
                MessageBoxButtons.OK, MessageBoxIcon.Information);
        }
    }
}

11. Lỗi thường gặp & Cách xử lý

❶ Lỗi SSL khi kết nối SQL Server

Thông báo lỗiA connection was successfully established with the server, but then an error occurred during the login process. (provider: SSL Provider, error: 0 – The certificate chain was issued by an authority that is not trusted.)

Nguyên nhân: .NET 8 mặc định yêu cầu SSL certificate hợp lệ khi kết nối SQL Server.

Giải pháp: Thêm TrustServerCertificate=True vào connection string.

❷ Scaffold không nhận lệnh (PowerShell ExecutionPolicy)

Thông báo lỗiFile cannot be loaded because running scripts is disabled on this system.

Giải pháp: Mở PowerShell với quyền Admin và chạy:

Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser

❸ Tên property sai sau Scaffold

Lỗi RuntimeSystem.ArgumentException: Column 'MaMH' does not belong to table.

Nguyên nhân: Code dùng MaMH nhưng Model scaffold sinh ra MaMh.

Giải pháp: Luôn mở file Mon.cs, KetQua.cs xem tên property chính xác trước khi viết code.

❹ Lỗi nchar — padding chuỗi

Lỗi logicKhông tìm thấy bản ghi dù đã nhập đúng / So sánh không khớp.

Nguyên nhân: SQL Server lưu nchar(6) luôn đủ 6 ký tự, chuỗi ngắn hơn được đệm khoảng trắng. Ví dụ: "LTW" được lưu thành "LTW ".

Giải pháp: Luôn dùng .Trim() khi so sánh, và .PadRight(6) khi Insert:

// So sánh: luôn Trim() cả hai vế
db.Mons.Any(m => m.MaMh.Trim() == maMh.Trim())

// Insert nchar(6): padding đủ độ dài
new KetQua { MaMh = maMh.PadRight(6) }

❺ Form con không vào MDI

Triệu chứngForm con hiện ra ngoài ứng dụng, không nằm trong frmMain.

Nguyên nhân: Quên set formCon.MdiParent = this trước khi .Show().

// SAI — Form bay ra ngoài
new frmSinhVien().Show();

// ĐÚNG — Form nằm trong MDI
var f = new frmSinhVien();
f.MdiParent = this;
f.Show();

❻ CSV không hiện tiếng Việt trong Excel

Triệu chứngMở file CSV bằng Excel, tiếng Việt bị lỗi font (?????? hoặc Ã )

Nguyên nhân: Thiếu BOM (Byte Order Mark) UTF-8 — Excel cần ký tự này để nhận biết encoding.

// Phải ghi BOM ở đầu file
sw.Write("\uFEFF");  // ← dòng này TRƯỚC khi ghi bất kỳ nội dung nào
sw.WriteLine("Môn học,Số tiết,Điểm,Xếp loại");
